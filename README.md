# trading-tom

A paper-trading simulation bot for US stocks with autonomous strategies and a trading-platform dashboard.

## Quick Start

```bash
./run.sh start
```

See `./run.sh help` for all commands.

## Development

| Command | Description |
|---------|-------------|
| `./run.sh start` | Start all services |
| `./run.sh stop` | Stop all services |
| `./run.sh logs` | Tail logs |
| `./run.sh test` | Run tests |
| `./run.sh shell` | Open shell in main container |
